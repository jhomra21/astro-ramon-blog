# Product Overview

Juan.dev is a personal blog and portfolio website showcasing software development work and expertise. The site serves as a professional presence with the following key features:

## Core Features
- **Personal Blog**: Technical articles and insights built with MDX for rich content
- **Portfolio**: Showcase of development projects and skills
- **Mini Flight Tracker**: Interactive flight tracking application demonstrating real-time data handling
- **Responsive Design**: Mobile-first approach ensuring accessibility across all devices

## Target Audience
- Potential employers and clients
- Fellow developers and tech community
- Visitors interested in technical content and projects

## Brand Identity
- Clean, professional aesthetic with dark theme support
- Performance-focused with optimized loading and SEO
- Interactive elements using smooth animations (Framer Motion)
- Modern typography and iconography (Lucide icons)

## Deployment
- Hosted on Cloudflare Pages at `juanrmb.pages.dev`
- Static site generation for optimal performance and security
