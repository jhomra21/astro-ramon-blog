I built a simple yet powerful web application that allows users to convert video files into GIF format directly in their browser. The project showcases modern web technologies and a focus on user experience.

## Features

- Browser-based video to GIF conversion
- No file size limits
- Fast processing
- Clean, minimal interface
- Open source

## Tech Stack

The project is built using modern web technologies and follows best practices for performance and user experience. You can check out the source code on [GitHub](https://github.com/jhomra21/vid-to-gif-converter).
