Explore the world of colors with 'What Color Is This', a web application that identifies colors from images. Simply upload an image, and the app will provide you with the exact color name and code.

## Features

- Accurate color identification
- Supports multiple image formats
- User-friendly interface
- Fast and reliable

## Tech Stack

The application is built using modern web technologies to ensure a seamless user experience. Check out the live version at [whatcoloristhis.pages.dev](https://whatcoloristhis.pages.dev/).
